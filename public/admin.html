Configure one receiving wallet per network. Users are guided to the correct wallet when they stake. All funds land in your wallet automatically.

🟡 BSC / BNB Chain Wallet BEP-20

Receives USDT, BNB, CAKE and all BSC tokens. This is your MetaMask 0x address on BSC network.

🔵 Ethereum Wallet ERC-20

Receives USDT ERC-20 on Ethereum mainnet. Usually the same 0x address as BSC — just a different network.

🔴 Tron Wallet TRC-20

Receives USDT TRC-20. Get this from Trust Wallet → Tron account. Starts with T.

🟠 Bitcoin Wallet BTC

Receives native BTC. Get this from Trust Wallet → Bitcoin account. Starts with 1, 3, or bc1.

💰 Commission Wallet

Withdrawal fees are sent here. Can be same as BSC wallet.

How it works: Users pick their network when connecting. BSC and ETH users send automatically via MetaMask. TRX and BTC users see your address to send manually, then submit their tx hash for your review.
